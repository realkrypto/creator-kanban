# UGC Leads Dashboard

A Kanban-style dashboard for managing UGC creator leads.

## Features
- **Kanban Board**: Drag and drop leads between acquisition stages.
- **Filtering**: Filter leads by game.
- **Sorting**: Sort leads by followers or name.
- **Premium Design**: Dark mode with glassmorphism effects.

## Setup

This project was created with Vite + React.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown (usually `http://localhost:5173`).

## Note on Environment
The project structure was generated manually as the `node` environment was not available during the setup process. All necessary files (`package.json`, `vite.config.js`, etc.) are included.
