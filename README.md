# UGC Leads Dashboard

A Kanban-style dashboard for managing UGC creator leads.

## Features
- **Kanban Board**: Drag and drop leads between acquisition stages.
- **Filtering**: Filter leads by game.
- **Sorting**: Sort leads by followers or name.
- **Premium Design**: Dark mode with glassmorphism effects.

## Setup

This project was created with Vite + React.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the URL shown (usually `http://localhost:5173`).

## How to Quit and Relaunch

### Quitting
1.  **Stop the Server**: In the terminal where the app is running, press `Ctrl + C`.
2.  **Close Terminal**: You can now safely close the terminal window or quit the application.

### Relaunching
To start working on the project again later:

1.  Open your **Terminal**.
2.  Navigate to the project folder:
    ```bash
    cd /Users/bryan.song/.gemini/antigravity/scratch/ugc-leads-dashboard
    ```
3.  Start the server:
    ```bash
    npm run dev
    ```
4.  Open the link shown (usually `http://localhost:5173`) in your browser.
